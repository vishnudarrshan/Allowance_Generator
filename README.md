# Employee Allowance Management System

A MERN stack application for managing employee allowances, shifts, and team tracking.

## Features

- ✅ Employee shift and allowance tracking
- ✅ Manager team management
- ✅ Admin panel for system management
- ✅ SSO authentication
- ✅ Allowance analytics and reporting

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Authentication**: JWT

## Local Development

### Prerequisites
- Node.js 18+
- MongoDB
